# Шпаргалка по Git

## Посмотреть историю коммитов
```bash
git log --oneline

## Посмотреть статус проекта
git status

## иницилизоровать проект
git init

## создать новую ветку 
git branch newbranch

## переключиться на эту ветку

git checkout newbranch

## Посмотреть историю тегов
git tag -l

## Удалить ветку локально
git branch -d имя_ветки

## Удалить ветку на GitHub
git push origin --delete имя_ветки

## Показать все ветки

git branch -a
